Development Notes:

- Use `yarn` for package management

To run in development:

1. `yarn install`
2. Setup `.env.local` per `.env.local.example`
3. `yarn docker:dev`
4. Go to `http://localhost:3000/`
5. `yarn docker:dev-stop` to stop containers
